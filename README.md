# NexLua 高性能框架

NexLua 是一个基于 LuaJIT、LuaJava 的高性能框架，针对安卓平台做了特定的优化。旨在替代 AndroLua 框架。

## 项目正在施工

欢迎提交 PR。QQ群: 954264081。TG: https://t.me/NexLua

## 开发路线

目前框架已实现基本功能，AndroLua+ 的基本功能都有。

- [x] 使用纯 C 重写了 LuaJava 桥
- [x] 实现 LuaJava 基本框架
- [x] 实现 LuaJava 库
- [x] 实现 import 库
- [x] 完善 LuaConfig 旨在实现类似动态 AndroidManifest 配置清单
- [ ] 完善 LuaJava 生态组件, LuaService 等
